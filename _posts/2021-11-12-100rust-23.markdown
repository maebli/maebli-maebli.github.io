## 23th Challenge
### Challenge

Solve [euler challange number 6](https://projecteuler.net/problem=6).

### Solution :white_check_mark:

	{% highlight rust %}
	fn main() {

	    let a = (1..101).fold(0,|acc,x|{acc+x*x});
	    let b = (1..101).fold(0,|acc,x|{acc+x});

	    let c = b*b-a ;

	    println!("{}",c);
	}

	{% endhighlight %}
